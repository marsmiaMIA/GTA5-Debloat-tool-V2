<h1 align="center">GTA5-Debloat (Improved / Future-Proof Version)</h1>

<p align="center">
This is an updated version of the original GTA5-Debloat script.<br>
Full credit for v1 goes to <a href="https://github.com/TonicGaro" target="_blank">TonicGaro</a>.
</p>

<h3 align="center">How to use:</h3>

<p align="center">
1. Place <b>debloatGTA5.bat</b> in the same folder where GTA V is installed.
</p>

<p align="center">
2. Run the batch file and confirm when prompted.
</p>

<h3 align="center">What's improved in this version?</h3>

<p align="center">
• More future-proof: instead of deleting only a fixed list of GTA Online DLCs, this version automatically removes <b>all MP DLC folders</b> except a few important ones that should stay.
</p>

<p align="center">
• Safer for Story Mode: some MP folders removed in the original version were found to cause issues with <b>Strangers and Freaks missions</b>
</p>

<p align="center">
• This version keeps key MP files that are still used by Story Mode, helping the game run properly while still reducing unnecessary bloat.
</p>

<p align="center">
Community discussion on why some files have to be kept for strangers and freaks to work:<br>
<a href="https://www.reddit.com/r/SteamDeck/comments/1i9d1cg/psa_for_those_who_still_play_gta_v_and_want_to/" target="_blank">
Reddit post discussing GTA V debloat issues and safe file removal
</a>
</p>

<h3 align="center">Things to know:</h3>

<p align="center">
• BattlEye and GTA Online content will be removed.
</p>

<p align="center">
• This is intended for Story Mode players who want less storage use.
</p>

<p align="center">
• When launching GTA V, you may see a BattlEye error.
To fix this, open Rockstar Games Launcher settings and disable BattlEye.
</p>
